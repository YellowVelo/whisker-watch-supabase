Score Scale Every pet starts each scored day at: 10/10 The score displayed to users must always be formatted as: {score}/10 Examples: 10/10 9/10 7/10 4/10 Do not display: 100 97 85 Stable Declining Monitor ________________________________________ 3. Health Attributes Included in Score Only the following five Health Attributes affect the 10-point Health Score: 1. Appetite 2. Water 3. Bathroom 4. Stool 5. Vomiting These are the only attributes that can deduct points from the daily Health Score. ________________________________________ 4. Wellbeing Attributes Excluded from Score The following Wellbeing Attributes do not affect the 10-point Health Score: 1. Energy 2. Mobility 3. Breathing 4. Itching They are displayed as directional chips only. ________________________________________ 5. Weight Weight is separate from the Health Score. Weight must display as its own chip and must not affect the 10-point Health Score. Weight trend is calculated versus the most recent prior weight entry. ________________________________________ Scoring Rules 6. Daily Starting Score Each pet starts each day at: 10 If the owner marks the day as normal, the score is: 10/10 If the owner skips the day, no Health Score is created for that date. Skipped days are unknown, not normal. ________________________________________ 7. Deduction Rules Each negative observation creates a deduction. Each Health Attribute can deduct a maximum of: 2 points per day Total deductions are subtracted from 10. Example: Starting score: 10 Total deductions: 3 Displayed score: 7/10 Minimum score: 0/10 Maximum score: 10/10 ________________________________________ 8. Weighted Deduction by Option Position Deduction is based on how negative the selected answer is. For each Health Attribute: • Normal / baseline option = 0 deduction • Mild negative option = 1 deduction • More significant negative option = 2 deductions • Maximum deduction per Health Attribute = 2 If an attribute has four negative choices: • First negative choice = 1 point deduction • Second negative choice = 1 point deduction • Third negative choice = 2 point deduction • Fourth negative choice = 2 point deduction The exact mapping must be configured per observation option, not inferred only by array position, so product can adjust mappings without rewriting scoring logic. ________________________________________ 9.
